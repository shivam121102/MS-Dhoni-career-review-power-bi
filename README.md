# MS Dhoni Career Review – Power BI Dashboard

This project analyzes the cricket career of **MS Dhoni**, one of the most successful captains in cricket history.

Using **Power BI**, the dashboard visualizes Dhoni’s career statistics, performance trends, and achievements across different formats of the game.

## About MS Dhoni

Mahendra Singh Dhoni is a legendary Indian cricketer and former captain of the Indian national team.  
He led India to major ICC victories including:

- ICC T20 World Cup (2007)
- ICC Cricket World Cup (2011)
- ICC Champions Trophy (2013)

He is widely known for his calm leadership style and finishing ability in limited-overs cricket.

## Project Objective

The goal of this project is to analyze Dhoni’s career performance using data visualization and extract meaningful insights from cricket statistics.

## Key Insights Analyzed

- Total runs scored across formats
- Batting average and strike rate
- Performance by year
- Match statistics and trends
- Career milestones and achievements

## Dashboard Features

- Interactive Power BI visuals
- Career statistics overview
- Performance trend analysis
- Clean and easy-to-understand charts

## Tools & Technologies

- Power BI
- Excel / Dataset
- Data Visualization
- Data Analysis
